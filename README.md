# okuban.nvim

> 奥 (oku = deep/inner) + kanban — "deep kanban" or "inner board"

A Neovim plugin that brings GitHub Projects v2 kanban boards directly into your editor.

## Vision

View and interact with your GitHub Projects as an ASCII kanban board without leaving Neovim. Open an overlay, see your columns and issues, click through to read issue details — all from the terminal.

## Status

🚧 **Early Development** — Not yet functional

## Planned Features

- [ ] Display GitHub Projects v2 as ASCII kanban board
- [ ] Navigate columns and cards with keyboard
- [ ] View issue details in floating window
- [ ] Move cards between columns
- [ ] Create new issues directly from the board
- [ ] Real-time sync with GitHub

## Why?

Existing solutions either:
- Are local file-based kanbans (no GitHub sync)
- Provide GitHub integration but no kanban view (octo.nvim)
- Import GitHub Projects as one-off snapshots (taskell)

okuban.nvim aims to be the missing piece: a live, interactive GitHub Projects kanban inside Neovim.
